## Proteno Tamil Dataset

This dataset contains 30, 000 sentences for text normalization.  There are two Python pickled lists containing tokenized unnormalized sentences (unnorm_list.pkl) and their corresponding tokenized normalizations (norm_list.pkl). First 60% of the data was used for training Proteno and the the models were evaluated on the other 40%.
